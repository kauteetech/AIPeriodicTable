RAG Pattern

The RAG pattern combines Embeddings (Em) → Vector Database (Vx) → Retrieval (Rg) to ground LLM responses in company data, making answers more accurate and contextual.

Agentic Loop

Agent thinks and plans a task, calls functions/tools to act, observes results, and repeats until the goal is achieved.

Agentic Pattern

The agentic loop combines Agent (Ag) → Function Calling (Fc) → Observation to create autonomous, goal-seeking behavior.
